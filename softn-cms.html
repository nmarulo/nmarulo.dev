Descripción:

SoftN CMS es un sistema de gestión de contenido con el que puedes crear sitios web dinámicos e interactivos, con una interfaces amigable e intuitiva que se adapta a cualquier tipo de resolución. Desarrollado en PHP como lenguaje de programación y MySQL como gestor de base de datos.

Enlace:

https://github.com/nmarulo/softn-cms

Demo: http://cms.softn.red/ user y pass: admin

Login: http://cms.softn.red/login

Instalación

Por el momento la instalación se realiza de forma manual.

En el fichero "app/config.php" configurar los datos de conexión a la base de datos, cambiar "root" por el usuario y la contraseña correspondiente:

define('DB_USER', 'root');

define('DB_PASSWORD', 'root');

Abrir el script SQL "softn_cms.sql" buscar y reemplazar la siguiente ruta "http://localhost/ProyectosWeb/htdocsxampp/SoftN-CMS/" con la ruta tu localhost.

Por ultimo ejecutar el script "softn_cms.sql", por ejemplo, usando PHPMyAdmin.

Acceder usando el usuario "admin" y contraseña "admin" o crear un usuario.

Preguntas

¿Cuales son los requisitos?
- PHP: 5.6.21 o superior
- MySQL: 5.6.25 o superior
- PDO PHP Extension
- Apache con mod_rewrite
¿Por qué estas realizando este proyecto existiendo tantos CMS?
- Porque quiero mejorar mis conocimientos sobre el desarrollo web y practicar lo que voy aprendiendo.
¿Cuanto tiempo llevas trabajando en este proyecto?
- En la versión 0.3: 2 meses.
¿Este proyecto empezó desde 0?
- Si.
¿Cuantos años de experiencia tienes en el desarrollo web?
- Poco más de un año.
¿En que idioma esta el proyecto?
- Todos los mensajes y comentarios de la aplicación están en Español. Mas adelante pretendo que pueda cambiar de idioma de los mensajes mostrados en la aplicación.
¿Donde puedo ver el proyecto en linea?
- Temporalmente aloje la aplicación en http://cms.softn.red/ (usuario y contraseña "admin").
